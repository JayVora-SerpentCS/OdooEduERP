School Fees Management

This module provides the mention features:
1. Student Fees Management.
2. Student Fees Receipt.
3. Student Fees structure.
4. Student Fees Register.

Fees Structure

Select Fees Structure Name,Code of the fees structure and User can add multiple fees pay lines in this tab.

Fees Receipt

User can create a fees receipt for perticular student and print fees receipt from here.

Fees Register

User can create Fees Register and print the Register from here.
