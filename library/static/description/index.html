Library Management

This module provides features to manage library information like Books, Publishers, Authors and Book Tracking.

Book Information

Librarian can create and update book details.

Book Request.

Student and Teacher can request for existing and new book.

Fine is calculated on the book if it is returned late than due date and when book is lost.

Invoice is generated of the student.

Student can view his own invoice.
